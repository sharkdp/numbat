# IDE / editor integration

There is syntax highlighting support for the following IDEs / text editors:

- [Vim](https://github.com/sharkdp/numbat/blob/main/assets/numbat.vim)
- [VS Code](https://github.com/sharkdp/numbat/tree/main/vscode-extension)
- [Sublime Text](https://github.com/sharkdp/numbat/blob/main/assets/numbat.sublime-syntax)
- [Helix (tree-sitter)](https://github.com/irevoire/tree-sitter-numbat)
